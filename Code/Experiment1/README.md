# Experiment 1: 单机串口实验 (Single-Device Serial Test)

## 实验介绍
本实验是整个项目的起点，旨在验证开发环境、硬件连接（自环）以及基础的串口收发功能。实验在单台计算机上进行。

## 实验内容
1.  **自环测试**: 将 TTL 模块的 TX 接到自己的 RX，验证自己发送的数据自己能否收到。
2.  **基准测试**: 测试设备的最大稳态传输速率。
3.  **可靠性测试**: 发送长文本，检测是否有位翻转或丢包现象。

## 运行方法

### 1. 硬件连接
- 将 USB-TTL 模块插入电脑。
- **自环连接**: 使用杜邦线将模块本身的 TX 引脚连接到 RX 引脚。

### 2. 环境准备
```bash
# 激活环境
conda activate cnetwork
# 安装依赖
pip install pyserial
```

### 3. 运行程序
```bash
python Code/Experiment1/main.py
```

## 预期结果
程序应能够显示“发送成功”并立即收到相同的内容。如果发生乱码或超时，请检查接线和波特率设置。
